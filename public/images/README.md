To add your profile image to the portfolio:

1. Save your profile image as: profile.jpg
   Location: public/images/profile.jpg

2. The About.tsx component has been updated to display:
   <img src="/images/profile.jpg" alt="Parth Azad - Full Stack Developer" />

3. Image specifications:
   - Recommended size: 1000x1000px or larger
   - Format: JPG, PNG, or WebP
   - File size: < 500KB for optimal performance
   - Aspect ratio: Square (1:1) works best with the circular design

4. After adding the image:
   - Test locally: npm run dev
   - The image will display in the About section
